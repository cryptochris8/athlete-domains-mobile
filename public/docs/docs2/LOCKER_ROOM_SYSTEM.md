# Locker Room System

## Overview
The Locker Room is the customization hub.

## Features
- Select active avatar
- Equip cosmetics
- View inventory
- Preview locked items
- Show rarity
- Show upgrade level

## Avatar Layers
### Base Avatar
- Identity
- Stats
- Rarity

### Cosmetics
- Visual customization

## Cosmetic Slots
### V1 (Launch)
- Shirt / Jersey
- Shoes

### V2 (Future)
- Socks
- Pants / Shorts
- Accessories
- Hats

## Goals
- Easy customization
- Encourage purchases
- Support future expansion
